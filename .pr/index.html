Feature 1: Auto-Discovery Scan

# Before: Manual repo configuration
lxa board config repos add owner/repo1
lxa board config repos add owner/repo2
lxa board scan

# After: One command auto-discovers repos
lxa board scan --user jpshackelford --since 30 --verbose

# Sync config with private GitHub Gist
